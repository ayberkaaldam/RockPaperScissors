How To Play

Hello Everyone ! This is classic rock , paper , scissors game but there are some different points. Let's talk about these changes. We have three different character :

You can select your character right top corner character selection button. Every character has special theme for themselves and also change gameplay. For example you select Rock for character. If you select rock in game and win you're going to get +2 points but if you lose ,your oppenent going to take +2 points. If you select paper or scissors in rock character you can win or lose 1 point.Good Luck Have Fun !
